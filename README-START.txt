========================================
  SD Quick UI - 빠른 시작 가이드
========================================

🎨 AI로 그림을 그려주는 프로그램입니다!

📦 설치 방법 (3단계)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. install.bat 더블클릭
     → 자동으로 설치됩니다 (10-30분 소요)

  2. 모델 다운로드
     → https://huggingface.co/runwayml/stable-diffusion-v1-5
     → v1-5-pruned-emaonly.safetensors 다운로드 (약 4GB)
     → stable-diffusion-webui\models\Stable-diffusion\ 폴더에 넣기

  3. start.bat 더블클릭
     → 브라우저가 자동으로 열립니다!

🚀 실행 방법
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  매번 사용할 때:
  → start.bat 더블클릭

  두 개의 검은 창이 열립니다 (닫지 마세요!)
  브라우저: http://localhost:5173

⚠️ 문제 발생 시
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. "Git이 설치되어 있지 않습니다"
     → https://git-scm.com/download/win 설치

  2. "Node.js가 설치되어 있지 않습니다"
     → https://nodejs.org/ 설치 (LTS 버전)

  3. 기타 문제
     → INSTALLATION-KR.md 참고
