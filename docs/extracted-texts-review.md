## 🔧 개선 제안

### 스크립트 개선 (선택사항)
현재 스크립트는 **충분히 사용 가능**하지만, 더 나은 결과를 위해:

```javascript
// 개선 사항 1: 코드 블록 필터링 강화
function isCodeBlock(text) {
  return text.includes('function') ||
         text.includes('const ') ||
         text.includes('import ') ||
         text.length > 200 ||
         text.split('\n').length > 5
}

// 개선 사항 2: 스타일 블록 제외
if (text.includes('background:') || text.includes('border:')) {
  // Skip CSS
}

// 개선 사항 3: 주석만 있는 텍스트 제외
if (text.trim().startsWith('//') || text.trim().startsWith('/*')) {
  // Skip comments
}
```

**예상 효과**:
- 파일 크기: 981KB → 약 100KB
- 유용한 텍스트 비율: 81% → 95%+

**필요성**: 낮음 (현재로도 충분)
